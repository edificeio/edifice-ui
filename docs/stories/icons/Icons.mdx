# Common Icons

To use this set of icons, import `@edifice.io/icons` into your code:

- `import { AddUser, AlertCircle, Check } from "@edifice.io/icons;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <button>
      <AddUser /> Add a new user
    </button>
  );
}
```
