# Navbars Icons

To use this set of icons, import `@edifice.io/icons/nav` into your code:

- `import { NeoAssistance, Search, MyApps } from "@edifice.io/icons/nav;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <nav>
      <MyApps />
    </nav>
  );
}
```
