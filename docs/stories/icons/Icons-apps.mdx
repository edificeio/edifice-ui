# Apps and Connectors Icons

This set of icons uses a SVG Sprite and should already be included with `edifice-bootstrap`.

## Usage

Use this set of icons with the `AppIcon` Component.

```jsx
import { AppIcon } from '@edifice.io/components';

export default function App() {
  return (
    <AppIcon
      app={{
        address: '/blog',
        icon: 'blog-large',
        name: 'Blog',
        scope: [],
        display: false,
        displayName: '',
        isExternal: false,
      }}
      size="48"
    ></AppIcon>
  );
}
```
