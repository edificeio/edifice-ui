# Reactions Icons

To use this set of icons, import `@edifice.io/icons/audience` into your code:

- `import { Great, GreatCounter } from "@edifice.io/icons/audience;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <div>
      <Great />
      <GreatCounter />
    </div>
  );
}
```
